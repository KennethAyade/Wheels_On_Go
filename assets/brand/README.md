# Brand Assets

- `logo/`: Primary and secondary app logos. Use PNG or SVG. Name files as `logo-primary.png`, `logo-inverted.png`, `logo-mono.png`.
- `icons/`: App icons, favicons, and small marks. Name files by size/context, e.g., `icon-1024.png`, `favicon-32.png`, `monogram.png`.
- Keep source files versioned (no editable binaries outside this folder). If exporting variants, include suffixes like `-light`, `-dark`, `-outline`.
- Do not commit licensed fonts or vendor-owned assets here.
